# goflyway, HTTP tunnel in Go

goflyway is an end-to-end encrypted HTTP/SOCKS5 proxy client/server written in golang, featuring:

1. TCP tunneling
2. TCP tunneling over WebSocket
3. Multiplex connections
4. Man-in-the-middle
5. UDP over TCP (SOCKS5)
